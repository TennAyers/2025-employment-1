ASP.NET CoreはデフォルトでHTTPS通信を使用します。初めて開発する際に、ブラウザから警告が出ないようにするため、一度だけ以下のコマンドを実行して開発用の証明書を信頼させてください。

証明書を信頼させる:
ターミナルで以下のコマンドを実行します。
dotnet dev-certs https --trust

アプリケーションを起動する:dotnet run
http://localhost:5147

Dataフォルダーを新規作成して、そこにApplicationDbContext.csを追加して
using Microsoft.EntityFrameworkCore;
using _2025_employment_1.Models;

namespace _2025_employment_1.Data 

{
    public class ApplicationDbContext : DbContext
    {
        public ApplicationDbContext(DbContextOptions<ApplicationDbContext> options)
            : base(options)
        {
        }

    }
}

上記内容を記述するこの際にアプリケーション名を挿入する今回は”_2025_employment_1”これ
